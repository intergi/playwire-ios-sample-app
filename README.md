# playwire-ios-sample-app

## PlaywireSDKApps

This workspace was created using the next tech stack:

- iOS target: 11.0+
- Languages: Swift, Objective-C
- Cocoapods

### Install pods

Open terminal and run next command

```sh
pod install
```
### Run demo app  

Open `PlaywireSDKApps.xcworkspace`, select required scheme and run.
